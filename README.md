# ⌗ Pingo

Photo Bingo Generator

## Instructions

- **Pingo.html** is a standalone HTML file that allows you to create custom Bingo cards using your own images. Download it, or run it directly in your web browser. [Open Pingo](src/Pingo.html).
- **Upload Images:** Click the upload area and select images from your computer. You can upload any number of images. The file name becomes the image title in the Bingo cards.
- **Configure Settings:** Set the number of cards, grid size (rows/columns), title, and optional footer text.
- **Generate Cards:** Click the "Generate" button to create your Bingo cards. They will appear on the right.
- **Print:** Click the "Print" button to open the print dialog. Each card will print on a separate A4 page.
